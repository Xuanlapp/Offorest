# Sticker Batch Update Sheet Summary

- Da them che do chon nhieu item trong Sticker page bang checkbox tung dong.
- Da them cac nut thao tac nhanh:
  - Chon trang nay
  - Bo chon
  - Update da chon
- Luong update hang loat chi xu ly cac item da co master image (`base64`).
- Ket qua update duoc cap nhat theo tung item qua `uploadStatus` (`uploading`, `done`, `error`).
- Upload hang loat dung chung 1 lan resolve sheet (`sheetId`, `gid`) de giam loi va tang toc do.
