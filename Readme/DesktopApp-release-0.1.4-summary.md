# Offorest Desktop Release 0.1.4

## Thay doi

- Tang version ung dung tu `0.1.3` len `0.1.4` de tao ban dist moi nhat.
- Build lai frontend vao thu muc `app-dist` truoc khi dong goi Electron.
- Dong goi ban phat hanh Windows bang `electron-builder` vao thu muc `release`.

## Lenh da dung

```bash
npm run build
npm run dist
```

## Ket qua mong doi

- Tao bo cai moi trong thu muc `release` voi metadata version `0.1.4`.
- Cap nhat thu muc `win-unpacked` theo noi dung build moi nhat.
