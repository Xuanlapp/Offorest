# Sticker Nav Get Data Summary

- Da them ho tro `Sticker` tren thanh nav voi o nhap `Sheet URL` va nut `Get Data`.
- Navbar gio doc/luu `stickerSheetUrl` trong `localStorage` de giu lai link sheet.
- Khi o trang `/sticker`, nut `Get Data` tren nav se phat event `stickerGetData`.
- `StickerPage` da nghe event `stickerGetData`, nen luong lay du lieu duoc kich hoat tu nav nhu ban yeu cau.
